Create a clean and modern FAQ section for an event or conference website.

Add a large centered heading that says "Frequently Asked Questions". Below it, display four FAQ cards in a two-column layout on desktop and a single column on mobile.

Each card should have a white background, rounded corners, a subtle shadow, and a thin blue accent border on the left. Make the question bold and easy to read, with the answer in a smaller gray font. Keep enough spacing between the cards so the section feels open and uncluttered.

Use a simple white and blue color palette with a professional, minimal design. Add a smooth hover effect where the card slightly lifts and the shadow becomes a little stronger. The layout should be fully responsive and suitable for a modern conference or SaaS landing page.
